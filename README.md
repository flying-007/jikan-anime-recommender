## 📖 Overview

This project is an anime discovery and recommendation system developed in Python using the Jikan API. Users can search for anime titles, retrieve detailed information, and receive recommendations based on available anime metadata.

The goal of this project is to demonstrate API integration, data processing, and recommendation system fundamentals while creating a practical application for anime enthusiasts.

---

## ✨ Features

- 🔍 Search anime by title
- 📚 Retrieve detailed anime information
- ⭐ View ratings, genres, and synopsis
- 🎯 Get anime recommendations
- ⚡ Real-time data fetched from the Jikan API
- 🐍 Built entirely in Python
