<h1 align="center">🎌 Anime Search & Recommendation System</h1>

<p align="center">
A Python-based anime discovery application that uses the Jikan API to search for anime and provide recommendations.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>Search anime by title</li>
  <li>Fetch anime details using the Jikan API</li>
  <li>Generate anime recommendations</li>
  <li>User-friendly interface</li>
  <li>Real-time data retrieval</li>
</ul>

<h2>🛠️ Technologies Used</h2>

<ul>
  <li>Python</li>
  <li>Jikan API</li>
  <li>REST API Integration</li>
  <li>JSON Data Processing</li>
</ul>

<h2>🚀 Usage</h2>

<p>
Search for your favorite anime and discover similar recommendations based on Jikan API data.
</p>

<h2>📚 Learning Outcomes</h2>

<ul>
  <li>Working with REST APIs</li>
  <li>Handling JSON responses</li>
  <li>Building recommendation systems</li>
  <li>Python application development</li>
</ul>
